<p>
Open media platform built with ExpressJS, AngularJS and TypeScript. Features user authentication, full CRUD for articles, and a commenting system with edit/delete support.
</p>

# To start app
Open a terminal and type:
- cd server <br />
- npm i <br />
- npm start <br />

Open a second terminal and type:
- cd client
- npm i
- ng serve

# Requirments
- npm
- node
- mongodb
- angular cli
- typescript
